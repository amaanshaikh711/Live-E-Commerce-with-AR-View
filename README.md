# Lusso Homes - Premium Furniture E-Commerce Website

![Lusso Homes Banner](screenshots/home-hero.png)

## üåü Overview

**Lusso Homes** is a premium e-commerce platform designed for luxury furniture. The website offers a sophisticated shopping experience with a focus on aesthetics, user experience, and modern design. Built with vanilla JavaScript and CSS, it delivers high performance without the overhead of heavy frameworks.

## ‚ú® Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktops, tablets, and mobile devices.
- **Dynamic Product Loading**: Products are loaded dynamically using JavaScript, allowing for easy updates and management.
- **Shopping Cart & Wishlist**: Functional cart and wishlist with local storage persistence.
- **Product Filtering**: Filter products by category (Sofas, Beds, Chairs, Tables, etc.).
- **Interactive UI**: Smooth animations, hover effects, and a modern carousel slider.
- **Contact Form**: Integrated contact form for customer inquiries.
